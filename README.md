# DND 5e 2024 翻译项目

> **Dungeons & Dragons 5th Edition (2024) 英文规则书 → 中文翻译**

---

## 项目简介

本项目旨在将 Dungeons & Dragons 第五版 2024 年修订版（俗称 5e 2024 或 "One D&D"）的核心规则书及配套出版物，从英文翻译为中文，为中文 DND 社区提供高质量的规则参考。

### 翻译范围

| 优先级 | 内容 | 状态 |
|--------|------|------|
| P0 | 2024 玩家手册（Player's Handbook） | 待开始 |
| P0 | 2024 地下城主指南（Dungeon Master's Guide） | 待开始 |
| P0 | 2024 怪物图鉴（Monster Manual） | 待开始 |
| P1 | 配套冒险模组 | 待开始 |
| P2 | 补充规则书（如 Xanathar、Tasha 等 2024 更新版） | 待开始 |

## 翻译流程

1. **预处理** — PDF 提取文本，确认章节结构
2. **术语确认** — 对照术语表，新增术语先登记
3. **初译** — 按章节翻译，保持一致性
4. **校对** — 逐句对照原文，核对规则数值
5. **润色** — 优化中文表达，统一风格
6. **终审** — 通读检查格式与排版

---

## 质量原则

- **准确性** — 规则术语必须精确，不产生歧义
- **一致性** — 全书术语统一，风格一致
- **可读性** — 中文表达自然流畅，符合 TRPG 阅读习惯
- **可追溯** — 每次修改有记录，术语变更可追溯

## 参与贡献

1. 阅读 [翻译规范](./style-guide/STYLE_GUIDE.md) 和 [术语字典](./glossary/GLOSSARY.md)
2. 在术语表中确认待翻译内容的术语
3. 按章节认领翻译任务
4. 提交翻译时附上自检清单
