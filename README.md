# DND 第三方文档翻译项目

> **基于 DND 规则的第三方英文文档 → 中文翻译**

---

## 项目定位

本项目专注于翻译基于 DND（Dungeons & Dragons）规则体系下的**第三方英文文档**，为中文 DND 社区提供更多优质的扩展内容参考。

**⚠️ 重要声明：本项目不翻译官方规则书内容。** 官方核心规则书（如《玩家手册》《地下城主指南》《怪物图鉴》等）及其配套出版物版权归 Wizards of the Coast LLC 所有，不在本项目翻译范围内。

### 翻译范围

| 类别 | 内容举例 | 说明 |
|------|----------|------|
| 第三方扩展 | 社区创作的规则扩展、变体规则 | 基于 DND 规则的第三方内容 |
| 第三方冒险 | 独立作者发布的冒险模组 | 非官方的冒险剧本 |
| 第三方设定 | 世界设定、战役设定集 | 社区或第三方发布的设定资料 |
| 工具与资源 | 第三方工具文档、资源指南 | 辅助跑团的工具类文档 |
| 社区指南 | 第三方攻略、指南、参考 | DM/玩家社区创作的内容 |

## 翻译流程

1. **选题评估** — 确认文档为第三方内容，评估翻译价值与版权合规性
2. **预处理** — 提取原文文本，确认文档结构
3. **术语确认** — 对照术语表，新增术语先登记
4. **初译** — 按章节翻译，保持一致性
5. **校对** — 逐句对照原文，核对规则数值
6. **润色** — 优化中文表达，统一风格
7. **终审** — 通读检查格式与排版

---

## 质量原则

- **准确性** — 规则术语必须精确，不产生歧义
- **一致性** — 全书术语统一，风格一致
- **可读性** — 中文表达自然流畅，符合 TRPG 阅读习惯
- **可追溯** — 每次修改有记录，术语变更可追溯

## 参与贡献

1. 阅读 [翻译规范](./style-guide/STYLE_GUIDE.md) 和 [术语字典](./glossary/GLOSSARY.md)
2. 确认待翻译内容属于第三方文档，不涉及官方规则书
3. 在术语表中确认待翻译内容的术语
4. 按章节认领翻译任务
5. 提交翻译时附上自检清单
