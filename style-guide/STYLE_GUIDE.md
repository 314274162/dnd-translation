# DND 5e 2024 翻译规范

> 版本：v1.0  
> 适用范围：DND 5e 2024 核心规则书及配套出版物  
> 最后更新：2025年

---

## 一、总则

### 1.1 翻译目标
- **准确性优先**：术语翻译必须准确传达原文规则含义，避免歧义
- **一致性**：同一术语在全书中翻译必须统一
- **可读性**：中文表达流畅自然，符合中文角色扮演游戏的习惯用语
- **尊重传统**：优先采用中文DND社区已广泛接受的译法

### 1.2 参考依据
1. DND 5e 2014 中文版（如存在官方或民间译本）
2. 中文TRPG社区通用译法（纯美苹果园、DND吧等）
3. 奇幻文学翻译惯例

## 二、格式规范

### 2.1 排版规则

| 规则 | 说明 |
|------|------|
| 中英文混排间距 | 中文与英文、数字之间加半角空格 |
| 标点符号 | 中文正文使用全角标点；英文术语、数字使用半角标点 |
| 引号 | 中文使用「」或""，英文术语保留 "" |
| 书名 | 使用《》包裹 |
| 游戏术语 | 首次出现或需强调时使用**粗体**标记 |

### 2.2 数字与单位

| 场景 | 规范 | 示例 |
|------|------|------|
| 游戏数值 | 保留阿拉伯数字 | 1d8、+2、15尺 |
| 距离单位 | feet → 尺 | 30 feet → 30尺 |
| 重量单位 | pound → 磅 | 10 pounds → 10磅 |
| 范围描述 | 保留数字+单位 | 120-foot range → 120尺射程 |

### 2.3 骰子表达式
- 保留原始格式：`XdY+Z`（如 2d6+3、1d20）
- 优势/劣势：advantage → 优势，disadvantage → 劣势
- 熟练加值：Proficiency Bonus → 熟练加值（简称 PB）

### 2.4 章节与标题
- 章节编号保留原文格式（如 Chapter 1、Part 2）
- 标题层级与原文保持一致
- 侧边栏/注释使用引用块格式

## 四、种族与职业名称

### 4.1 翻译原则
- 优先使用社区已有共识译名
- 新种族/职业名称需在术语表中登记
- 译名应体现文化内涵，避免纯音译

### 4.2 格式要求
- 种族名称：二字或三字词为佳
- 职业名称：尽量使用二字词
- 子职（Subclass）：以「道途/学派/领域/誓言/契约」等词区分不同职业的子职业体系

---

## 五、怪物与NPC名称

### 5.1 翻译原则
1. **保留原名**：经典DND怪物（如 Beholder → 眼魔、Mind Flayer → 灵吸怪）
2. **意译优先**：描述性名称用意译（如 Fire Giant → 火巨人）
3. **音译+注释**：无法意译的名称使用音译，首次出现附原文
4. **首字母大写**：怪物名称作为专有名词处理

## 六、专有名词

### 6.1 世界设定
- 地名：已约定俗成的使用中文（如 Waterdeep → 深水城）
- 人名：首次出现保留原文括号（如 崔斯特·杜垩登（Drizzt Do'Urden））
- 组织名：意译优先，保留缩写（如 Harpers → 竖琴手同盟）

## 七、翻译流程

### 7.1 标准流程
1. **预处理**：PDF → 可编辑文本，确认章节结构
2. **初译**：按章节分配，使用术语表保证一致性
3. **校对**：对照原文逐句检查，重点核对规则数值
4. **润色**：优化中文表达，统一行文风格
5. **终审**：通读全文，检查格式和排版

### 7.2 质量检查清单
- [ ] 术语翻译与术语表一致
- [ ] 游戏数值准确无误
- [ ] 骰子表达式格式正确
- [ ] 表格数据完整
- [ ] 交叉引用有效
- [ ] 中英文混排间距正确
- [ ] 标点符号使用正确

## 附录：常见问题

**Q: 法术名应该意译还是音译？**  
A: 意译优先。如 Fireball → 火球术，Magic Missile → 魔法飞弹。经典法术保留约定俗成的译法。

**Q: "level" 如何区分翻译？**  
A: 角色等级 → 等级（如 level 5 → 5级）；法术等级 → 环（如 3rd-level spell → 三环法术）。

**Q: 伤害类型是否需要统一？**  
A: 是的，所有伤害类型必须统一翻译，详见术语表。
